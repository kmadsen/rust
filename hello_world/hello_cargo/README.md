# Cargo

Cargo is a Rust build system and package manager.

## Create

$ cargo new hello_cargo
$ cd hello_cargo
$ cargo check

## Build

$ cargo build

## Run

$ ./target/debug/hello_cargo
$ cargo run

## Integration tests

This integrates with the hello_crate library.

$ cargo test

## Useful tools

$ cargo fmt
$ cargo fix
$ cargo clippy
